  • PrometheusMetricsFactory

    A bridge between PrometheusClient and swift-metrics. Prometheus types don’t map perfectly on swift-metrics API, which makes bridge implementation non trivial. This class defines how exactly swift-metrics types should be backed with Prometheus types, e.g. how to sanitize labels, what buckets/quantiles to use for recorder/timer, etc.
